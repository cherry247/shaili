# Minimalist Developer Portfolio (JSON-Powered)

A sleek, high-performance, and fully customizable portfolio template built with **Vanilla JS, CSS3, and HTML5**. This project is designed for developers who want a "classy" professional presence without the overhead of heavy frameworks. 

The entire site—content, structure, and even the color theme—is driven by a single `data.json` file.

---

## Quick Start

1.  **Fork** this repository to your own GitHub account.
2.  **Clone** your fork to your local machine.
3.  **Edit `data.json`**: Replace the placeholder text with your own details.
4.  **Add your photo**: Drop your profile image into the `assets/` folder.
5.  **Deploy**: Push your changes and enable **GitHub Pages** in your repository settings.

---

## Customization Guide

### 1. The Content (`data.json`)
The site is built to be "plug-and-play." You don't need to touch the HTML or CSS to make it yours.
* **Hero Section**: Update your `headline`, `availability` status, and `CTA` (Call to Action) links.
* **Experience**: Add your career history. The timeline will generate and animate automatically.
* **Projects**: Add as many projects as you like to the array; the grid layout adjusts to fit.
* **About**: Use the "One Paragraph" approach for a concise, professional bio.

### 2. The Visuals (Themes)
Inside `data.json`, you will find a `"theme"` object. You can completely change the aesthetic by swapping the hex codes:
* `--bg`: Background color (e.g., `#F5F2EC` for a paper feel).
* `--ink`: Primary text color.
* `--accent`: The highlight color used for buttons, tags, and icons.
* `--line`: The color of borders and dividers.

### 3. Your Profile Image
For the "classy" minimalist look:
1.  Navigate to the `assets/` folder.
2.  Upload your professional headshot.
3.  **Required Dimension**: For best results, crop your image to a square (**45x45 pixels**) or a 1:1 ratio.
